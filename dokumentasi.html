Versi 1.1.0 - Scanner Edition

1 Fitur Unggulan & Penggunaan

  • 100% Offline-First: Menggunakan Service Worker tingkat lanjut. Setelah dibuka pertama kali, aplikasi (termasuk scanner kamera) tetap bisa digunakan tanpa kuota internet.
  • Pemindai QR (Scanner) & Riwayat: Pindai QR code langsung dari kamera HP Anda. Hasil pindai (teks/URL/WiFi) otomatis disimpan ke "Riwayat Pindai" dan bisa dibagikan ulang kapan saja.
  • Kustomisasi Warna & Logo QR: Saat membuat QR, Anda bisa mengubah warna dasar barcode dan mengunggah gambar/logo untuk disisipkan di tengah QR. (Didukung fitur Error Correction Level H agar tetap bisa di-scan).
  • Ekspor/Cetak Sempurna & Quick Share: Hasil unduhan gambar QR dijamin tidak terpotong (Anti-Clipping). Tersedia tombol "Bagikan" (Quick Share) langsung dari halaman koleksi ke WhatsApp/Telegram.
  • State Persistence & Backup Excel: Ketikan Anda tak akan hilang walau web di-refresh (Auto-Save ke LocalStorage). Seluruh database bisa dicadangkan dan dipulihkan menggunakan file `.xlsx`.

2 Panduan Instalasi PWA (Icon & Splash Screen)

Agar aplikasi bisa di-install layaknya aplikasi Native (Android/iOS/PC), Anda WAJIB menyiapkan Icon Logo.

  1. Buat folder baru bernama assets sejajar dengan file index.html.
  2. Siapkan logo aplikasi (background solid direkomendasikan).
  3. Resize logo tersebut menjadi 2 ukuran persis:
    • 192x192 pixel. Simpan dengan nama icon-192x192.png
    • 512x512 pixel. Simpan dengan nama icon-512x512.png
  4. Masukkan kedua gambar tersebut ke dalam folder assets/.
  5. Jika berhasil, saat Anda membuka web via HP (Chrome/Safari), akan muncul notifikasi "Tambahkan ke Layar Utama" (Add to Home Screen).

3 Panduan Hosting & Upload

Aplikasi ini berbasis HTML statis, sehingga sangat mudah dan GRATIS untuk dihosting.

Opsi 1: GitHub Pages (Gratis & Disarankan)

  • Buat akun GitHub, lalu buat Repository baru.
  • Upload keempat file tersebut (index.html, manifest.json, sw.js, folder assets).
  • Masuk ke menu Settings > Pages di Repository tersebut.
  • Pada opsi 'Branch', ubah dari 'None' ke 'main', lalu klik Save.
  • Tunggu 1-2 menit, web Anda sudah live dan memiliki SSL HTTPS otomatis!

Opsi 2: cPanel / Shared Hosting

  • Login ke cPanel Hosting Anda. Buka File Manager.
  • Buka folder public_html (atau folder subdomain Anda).
  • Upload file index.html, manifest.json, sw.js, dan folder assets.
  • Aplikasi siap diakses melalui domain Anda. Pastikan domain Anda menggunakan HTTPS agar Service Worker (PWA Offline) maupun Kamera Scanner dapat bekerja dengan baik.
